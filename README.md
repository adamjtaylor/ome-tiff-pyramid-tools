# ome-tiff-pyramid-upgrade

Upgrade legacy OME-TIFF pyramids to the BioFormats 6 OME-TIFF pyramid format.

## Requirements

Python 3.7 or higher, or Python 3.6 with the `dataclasses` backport installed.
